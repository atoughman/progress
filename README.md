<h1 align="center">My React Habits Tracker App</h1>

This is a simple habits tracker app built with React. Users can add new habits, mark them as done or not done for the day, and track their streaks. The app also resets habits every 24 hours, so users can start fresh each day.

## Installation

1. Clone the repository to your local machine
2. Install dependencies by running `npm install`
3. Start the app by running `npm start`
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Usage

1. Enter a name and description for a new habit, and click "Add Habit"
2. Click the "Mark as done" button to mark a habit as done for the day
3. Click the "Mark as not Done" button to mark a habit as not done for the day
4. The streak count for each habit will update automatically based on whether it was marked done or not done for the day

## Technologies Used

- React
- Axios
- JSON Server
- SCSS
